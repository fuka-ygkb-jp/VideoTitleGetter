■概要
型番だけのエロ動画ファイル名を、より良いファイル名（動画タイトル）に変更する
Windowsアプリです。

エロ動画のファイル名から型番らしきものを抽出、それでNyaaTorrents(以後NT)を
検索し、より良いファイル名（タイトル）にリネームします。

Torrentファイルをダウンロードするとファイル名が型番しか無いことがあります。
これだと後で女優名やタイトルが分からないため、その後の自家発電業務に大きな支障を
きたすことがあります。ファック！!
NTを使い型番から正しいタイトルを求めることでエロ動画を整理整頓！
貴方の自家発電ライフを快適支援します。　Enjoy!


■使い方
１．リネームしたいファイルをドラッグ＆ドロップします。
２．「型番候補」欄に型番と思しき文字がセットされます。
　　これで良ければ右横の「検索」ボタンを押します。
３．運が良ければ「ファイル名候補」欄に検索結果が表示されます。
　　何も出ない場合は一致なしです。
　　「検索先」をGoogleにしてみたら何か見つかるかもしれません。
４．これだと思う物を一覧から選び、「ファイル名変更」ボタンを押します。
　　微調整する場合は「変更後ファイル名」欄を修正してから
　「ファイル名変更」ボタンを押します。


■インストール方法
圧縮ファイル内のファイルを適当なフォルダに解凍します。
そのフォルダの[VideoTitleGetter.exe]をダブルクリックすれば起動します。


■アンインストール方法
プログラムのあるフォルダを削除します。


■動作環境
Windows ＋ .Net Framework 4.5以降必須


■お断り
ある程度は動作テストしていますが、万が一貴方の大切なコレクションが消えてしまった
場合は…ごめんなさい。
大切なファイルは必ずバックアップを取ってからお使いください。
